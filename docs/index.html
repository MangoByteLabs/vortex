Python + PyTorch — Same task
import torch
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(2, 8), nn.ReLU(),
    nn.Linear(8, 1), nn.Sigmoid()
)
opt = torch.optim.Adam(model.parameters(), 0.01)
data = torch.tensor([[0,0],[0,1],[1,0],[1,1]], dtype=torch.float)
labels = torch.tensor([[0],[1],[1],[0]], dtype=torch.float)

for _ in range(1000):
    loss = nn.BCELoss()(model(data), labels)
    opt.zero_grad(); loss.backward(); opt.step()

with torch.no_grad():
    for x in data:
        print(model(x).item())

Why Vortex?

One language for AI, GPU computing, and cryptography. No Python glue. No C++ headers. No CUDA boilerplate.

⚡

Native GPU Compilation

Compiles through MLIR to PTX (NVIDIA) and AMDGCN (AMD). Write kernel functions and run them on the GPU with zero FFI.

🧠

Built-in Neural Networks

Linear, Conv2D, Transformer, LSTM, GRU layers as builtins. Train with Adam/SGD in one function call. No framework needed.

🎯

Type-Safe Tensors

Tensor autodiff with automatic backward pass. Reshape, broadcast, and matmul with shape checking at runtime.

🔄

Auto-Differentiation

Scalar and tensor autodiff tapes built in. Forward and backward AD for training loops, ODEs, and scientific computing.

🔗

Cryptography Native

secp256k1 elliptic curves, ECDSA/Schnorr signatures, SHA-256, field arithmetic, NTT, pairings, and ZK primitives.

🤖

MCP for AI Agents

Built-in Model Context Protocol server. AI agents can create, train, and query models via JSON-RPC. First-class agent support.
